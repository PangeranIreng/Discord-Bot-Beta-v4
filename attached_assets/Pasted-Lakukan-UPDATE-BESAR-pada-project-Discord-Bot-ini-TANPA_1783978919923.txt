5. HELP COMMAND
==================================================

Perbarui command

/help

Tambahkan seluruh command yang sudah ada.

Contoh:

🎫 Ticket

/cticket
/delcticket

🐞 Report

/cbug
/delcbug

🎵 BoomBox

/boombox
/setlimit
/setpremiumlimit

👑 Premium

/addpremium
/removepremium

📊 Monitoring

/panelstats

dan seluruh command lainnya.

Setiap command memiliki deskripsi singkat.

Contoh:

/cticket

Membuat Panel Open Ticket.

/delcticket

Menghapus seluruh Panel Ticket beserta Dashboard Ticket.

/cbug

Membuat Panel Report Bug & Feature.

/delcbug

Menghapus Panel Report Bug & Feature.

==================================================
6. DELCTICKET
==================================================

Tambahkan command baru

/delcticket

Owner Only.

Saat dijalankan:

✔ Menghapus Panel Ticket.

✔ Menghapus Dashboard Ticket Logs.

✔ Menghapus seluruh Button.

✔ Menghapus Message ID.

✔ Menghapus konfigurasi Ticket.

✔ Membersihkan konfigurasi database Ticket.

Bot mengirim:

✅ Ticket System berhasil dihapus.

==================================================
7. DELCBUG
==================================================

Tambahkan command

/delcbug

Owner Only.

Saat dijalankan:

✔ Menghapus Panel Report.

✔ Menghapus seluruh Button.

✔ Menghapus Message ID.

✔ Menghapus konfigurasi Bug Report.

✔ Membersihkan database konfigurasi.

Bot mengirim:

✅ Report Center berhasil dihapus.

==================================================

Pastikan:

• Tidak ada error.

• Tidak ada duplicate message.
• Tidak ada memory leak.
• Tidak ada spam message.

• Seluruh Button dan Dropdown tetap aktif setelah bot restart.
