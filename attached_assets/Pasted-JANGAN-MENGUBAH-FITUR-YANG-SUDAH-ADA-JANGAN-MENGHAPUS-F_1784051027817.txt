==================================================
2. OTOMATIS HAPUS PESAN LAMA
==================================================

Saat /premstats pertama kali dijalankan:

Cari embed Premium Monitoring lama.

Jika ditemukan:

hapus otomatis.

Jangan biarkan ada dua panel.

Hanya boleh ada SATU panel Premium Stats.

==================================================
3. COMMAND BARU
==================================================

Tambahkan command baru:

/premstats

Hanya Owner dan Developer.

Saat dijalankan:

Bot meminta Channel.

Setelah dipilih:

Bot membuat satu panel Premium Stats.

Selanjutnya panel hanya diedit.

Jangan mengirim pesan baru setiap update.

5. AUTO UPDATE
==================================================

Panel harus otomatis update ketika:

Premium expired.

Set Limit User.

Set Limit Role.

Limit direset.

Custom Limit berubah.

Sisa limit berubah.

Tidak perlu command ulang.

Bot hanya edit panel.

==================================================
6. USER EXPIRED
==================================================

User yang sudah expired:

Jangan ditampilkan lagi.

Jangan memenuhi panel.

Cukup kirim log ke Premium Logs.
