# Tambahkan Sistem Bug Report & Feature Request (Tanpa Mengubah Fitur Lama)

JANGAN menghapus, mengubah, atau merusak fitur yang sudah ada. Semua fitur Boombox, Open Ticket, Premium, Role Panel, Scanner, AI, Admin Panel, Database, dan Command lain WAJIB tetap berjalan seperti sebelumnya.

Tambahkan sistem baru bernama "Bug Report & Feature Request" dengan UI modern dan konsisten dengan tampilan Ticket System.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
SLASH COMMAND
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Buat Slash Command:

/cbug

Hanya Owner yang dapat menggunakan command ini.

Parameter:

• panel_channel (Channel)
→ Channel tempat Panel Bug & Feature dikirim.

• logs_channel (Channel)
→ Channel tempat semua laporan Bug & Feature masuk.

• developer_role (Role)
→ Role yang otomatis di-mention ketika ada laporan baru.

Saat command dijalankan:

- Simpan seluruh konfigurasi ke database.
- Jika panel sudah pernah dibuat, edit panel lama (jangan spam panel baru).
- Simpan Message ID panel agar tetap aktif setelah bot restart.
